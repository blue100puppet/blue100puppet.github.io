## Overview

A curated list of high-frequency **simplified Chinese characters** (汉字, *hànzì*) for absolute beginners. Focus on reading recognition first — these characters appear everywhere in daily life, signs, menus, and basic text.

Each entry includes: character, pinyin, English meaning, and a simple usage example.

## Stroke Order Basics

1. Top → Bottom: 人 — first stroke top, second bottom
2. Left → Right: 大 — left horizontal, then right cross
3. Horizontal → Vertical: 十 — horizontal first
4. Enclosure → Inside: 口 — draw box, then midpoint

## Learning Tips

**Radicals (部首, *bùshǒu*) hint at meaning:**
- 氵 (three drops water) → water: 水, 河, 海
- 木 (tree) → wood/plant: 林, 森, 树  
- 火 (fire) → fire/heat: 炎, 点, 热
- ⼈ (person) → person: 他, 你, 从

**Common components:**
- **女** (nǚ, woman) → 她, 好, 妈
- **子** (zǐ, child) → 学, 字, 孩
- **言** (yán, speech) → 说, 话, 语

## Study Strategy

- **Daily:** 10–15 new characters + review (use flashcards)
- **Write each 5–10 times** — stroke order matters!
- **Group by meaning** — not alphabetically (people → food → places)
- **Contextualize:** Note the radical and common words it appears in
